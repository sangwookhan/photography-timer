# PTimer

Native iPhone app bootstrap for the PTimer project.

## Stack

- Swift
- SwiftUI
- Xcode project

## Project Status

- iPhone SwiftUI app project created
- Local Git repository initialized
- Initial native build verified with `xcodebuild`
- iOS Simulator launch confirmed locally

## Getting Started

1. Open `PTimer.xcodeproj` in Xcode.
2. Select an iPhone Simulator.
3. Build and run the `PTimer` scheme.
