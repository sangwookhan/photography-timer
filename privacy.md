# Privacy Policy

Photography Timer does not require an account.

For the current 0.7 release scope:
- The app does not collect personal information.
- The app does not use advertising.
- The app does not sell user data.
- Timer, camera, film, and custom film settings are stored locally on the device unless the user explicitly exports or shares data in a future feature.

Use the support page or GitHub repository for questions.
